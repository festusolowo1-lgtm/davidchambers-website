# DAVID CHAMBERS — Legal Website

FastAPI + SQLite website for David Chambers Legal Practitioners, Abuja Nigeria.

## Deploy to Railway

1. Push ALL files in this folder directly to your GitHub repository root
2. Connect the repo to Railway
3. Railway auto-detects Python and builds

## Run Locally

```
pip install -r requirements.txt
python main.py
```

Server starts at http://localhost:8000
